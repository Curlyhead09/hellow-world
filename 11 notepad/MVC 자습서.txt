ASP.NET Core MVC 앱
	MVC 아키텍처 패턴은 앱을 모델, 뷰 및 컨트롤러의 세 가지 주요 구성 요소로 구분
		- Models : 비즈니스 논리는 모델
			데이터를 나타내는 클래스, 유효성 검사 논리를 사용 데이터에 대한 비즈니스 규칙 적용
			모델 상태를 검색하고 저장
		- Views : UI 논리는 뷰
			앱의 UI를 표시 
			일반적으로 UI는 모델 데이터를 표시	
		- Contollers : 입력 논리는 컨트롤러
			브라우저 요청을 처리
			모델 데이터를 검색
			응답을 반환
	 분리하면 다른 코드에 영향을 주지 않고 한 번에 한 구현 측면에서 작업할 수 있기 때문에 
	앱을 작성할 때 복잡성을 관리하는 데 도움
