AppData 폴더 - c:\Users\사용자명\AppData
	Application Data 약자
	비스타에 도입 (~ win10 사용 중)
	각각의 윈도우즈 사용자 계정에 존재
	응용 프로그램의 데이터 및 설정 내용을 저장
	탐색기 바로가기 - %APPDATA% 
	
	구조 
		Local
			단일 컴퓨터의 특정한 데이터가 저장 됨
			컴퓨터간에 동기화 되지 않음
			고유 데이터야만 하는 데이터나 매우 큰 파일들이 포함 됨 
				다운로드 캐시 파일, 기타 대용량 파일
		LocalLow
			Local 폴더와 동이하지만 제한된 보안 설정으로 실행되는(low integrity) 응용 프로그램을 위해 설계
				보호모드로 실행되는 인터넷 익스플로우는 LocalLow 폴더에만 접근
			일부 응용 프로그램에서는 기본 Local 폴더에 대한 접근 권한이 없기 때문에 사용 됨
		Roaming 
			PC가 프로필이 있는 도메인에 연결되어 있는 경우(회사, 기관 등) 컴퓨터와 컴퓨터간에 로밍데이터를 저장 
				북마크 및 기타 검색 데이터가 PC에서 PC로 이동
	
	
	도메인에 연결되어 있지 않은 경우 Roaming과 Local 폴더간에는 차이가 없음
		어떤 폴더에 저장 할지는 응용 프로그램 개발자에 의해 결정

ProgramData - c:\ProgramData (숨김파일)
	단일 설정이나 파일집합을 원할 경우 사용
		바이로스 백신 프로그램의 검사 로그와 설정을 유지하고 PC의 모든 사용자와 공유를 원할 경우
