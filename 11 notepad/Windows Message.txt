Windows 메시지	
	시스템은 메시지 형식의 창 프로시저에 입력을 전달
	메시지는 시스템과 애플리케이션 모두에서 생성
	시스템은 사용자 입력 이벤트에서 메시지를 생성
	시스템은 애플리케이션에서 가져온 시스템의 변경에 대한 응답으로 메시지를 생성
	애플리케이션은 작업을 수행하거나 다른 애플리케이션의 창과 통신 하도록 자체 창을 지시하는 메시지를 생성
	
	시스템은 창 프로시저에 메시지를 보냄 
	창 핸들은 메시지가 의도된 창을 식별
	시스템에서 이를 사용하여 메시지를 수신해야 하는 창 프로시저를 결정
	
	메시지 식별자는 메시지의 용도를 식별하는 명명된 상수
	창 프로시저가 메시지를 받으면 메시지 식별자를 사용하여 메시지를 처리하는 방법을 결정
	
	메시지 유형
		시스템정의 메시지
			시스템은 애플리케이션과 통신할 때 시스템 정의 메시지를 보내거나 게시
			메시지를 사용하여 애플리케이션의 작업을 제어하고 애플리케이션이 처리할 입력 및 기타 정보를 제공
			애플리케이션은 시스템 정의 메시지를 보내거나 게시할 수도 있음
			
			각 시스템 정의 메시지에는 고유한 메시지 식별자와 메시지의 목정을 나타내는 해당 기호 상수가 있음
			상수의 접두사는 메시지를 해석하고 처리할 수 있는 창 유형을 식별			
			
		애플리케이션 정의 메시지
			다른 프로세스에서 창과 통신하는데 사용할 메시지를 만들 수 있음
			자체 메시지를 만드는 경우 메시지를 수신하는 창 프로시저는 메시지를 해석하고 적절한 처리를 제공해야 함수를
			
			


Control.WndProc(Message) 메서드
	- Windows 메시지를 처리
	
	- 설명
		모든 메시지는 메서드를 WndProc 통해 필터링된 후 메서드로 PreProcessMessage 전송됨
			https://docs.microsoft.com/ko-kr/dotnet/api/system.windows.forms.control.preprocessmessage?view=windowsdesktop-6.0
		WndProc 메서드는 Windows WindowProc 함수와 같습니다. Windows 메시지 처리에 대한 자세한 내용은 WindowProc 함수를 참조
			https://docs.microsoft.com/en-us/windows/win32/api/winuser/nc-winuser-wndproc?redirectedfrom=MSDN
