가상 환경이란?
	가상 환경은 프로그램을 실행하는 데 필요한 모든 것의 자체 포함 복사본
	여기에는 Python 인터프리터와 프로그램에 필요한 모든 라이브러리가 포함
	가상 환경을 사용하면 프로그램이 올바른 버전 및 리소스에 액세스하여 올바로 실행되도록 할 수 있습니다.

	기본 워크플로
		컴퓨터의 나머지 부분에 영향을 주지 않는 가상 환경을 만듭
		가상 환경을 한 단계씩 실행하며 필요한 Python 버전 및 라이브러리를 지정
		프로그램을 개발
		
가상 환경 만들기
	venv 모듈을 호출하여 가상 환경을 만듦
		3.3 부터는 venv 모듈이 내장되기 때문에 별도 패키지 설치없이 파이썬만 설치되어 있으면 바로 가상 환경 구성이 가능
	모듈은 인수로 이름을 예상
		프로젝트를 유지하려는 디렉터리로 이동
		다음 명령을 사용하여 venv 모듈을 호출합니다. 이 명령은 운영 체제에 따라 약간씩 다름
		
	python -m venv env
	
가상 환경 활성화
	이 시점에서는 가상 환경이 있지만 아직 사용을 시작하지는 않았습니다. 
	가상 환경을 사용하려면 env 디렉터리에서 activate 스크립트를 호출하여 활성화해야 합니다. 
	Windows, Linux 및 macOS에서 활성화가 표시되는 방법은 다음과 같습니다.
		# Windows
			source env/Scripts/activate
	
	activate를 호출하면 프롬프트가 바뀝니다. 이제 다음 예제와 같이 프롬프트 앞에 (env)가 표시

패키지 설치
	pip를 사용하여 패키지를 설치	이 pip 명령은 PyPi(Python Package Index)를 사용하여 패키지를 가져올 위치를 파악 
	PyPi 웹 사이트를 방문하면 사용할 수 있는 패키지를 확인할 수 있습니다.
		pip install python-dateutil
	pip freeze를 실행할 수 있습니다. 이 명령은 터미널에 설치된 패키지의 목록을 생성

Python에서 부울 논리 사용
	학습 목표
		if, else, elif 문을 사용하여 다양한 조건에서 코드를 실행합니다.
		and 및 or 연산자를 사용하여 조건부 논리를 결합하고 더 복잡한 조건을 만듭니다.
	
	if’ 문 작성
		같음: a == b
		같지 않음: a != b
		보다 작음: a < b
		보다 작거나 같음: a <= b
		보다 큼: a > b
		보다 크거나 같음: a >= b
		
		a = 93
		b = 27
		if a > b:
			print("a is greater than b")
		elif a < b:
			print("a is less than b")
		else: 
			print ("a is equal to b")
