라이선스 파일 설정 우선 순위
	3.1.1.1. License Files Settings Precedence


1. ANSYSLI_SERVERS environment variable (환경변수)

2. If ANSYSLI_SERVERS is not set, use ANSYSLMD_LICENSE_FILE environment variable, replacing
the FlexNet port number with the default Licensing Interconnect port number of 2325.

	서버가 설치X ANSYSLMD_LICENSE_FILE 환경변수를 사용

3. Settings in the ansyslmd.ini file in the relative client licensing directory for ANSYSLI_SERVERS=
keyword
	

FlexNet settings have precedence in the following order:
1. ANSYSLMD_LICENSE_FILE environment variable
2. Settings in the ansyslmd.ini file in the relative client licensing directory for SERVER=keyword.
Note:
The FlexNet environment variable LM_LICENSE_FILE is not supported with the ANSYS
License Manager.
