웹개발 기초
Hosts 파일이란
	호스트 이름에 대응하는 IP 주소가 저장되어 있어서 도메인 이름 시스템(DNS)에서 주소 
	정보를 제공받지 않고도 서버의 위치를 찾게 해주는 파일
	
	역할
		호스트 이름에 대응하는 IP주소가 저장되어 있어서 도메인 이름 시스템에서 
		주소 정보를 제공받지 않고도 서버의 위치를 찾게 도와주낟
	장점
		인터넷 속터 향상, 리소스 사용을 줄임, 보안 문제 예방적 대처
	단점
		사이트 방문이 차단될 수 있음, 페이지 내에서 부분 차단된 경우 디자인.속도 문제
	위치
		C:\windows\system32\drivers\etc\hosts
	작성 원칙
		# : 주석
		각 항목은 한줄로 작성
		항목은 IP주소 + 호스트 이름 순서로 제한
		호스트 이름과 IP 주속의 간격은 최소한 1칸을 띄움

도메인(Domain) & IP 기본 개념
	ip - 어떤 홈페이지를 들어가기 위해 필요
		 2진수 32자리의 네트워크 주소
		 모든 아이피 주소를 외우기 힘들기 때문에 만든 것이 Domain주소 (예)www.naer.com)
	Domain Serve
		주소창에 domain 주소를 입력하면 실제 ip주소를 알려주는 역할		
			Root DNS Servers - 최상위 root dns 서버에서 하위 루트의 여러개의 dns 주소를 관리
				com DNS Servers
					~~.com
				org DNS Servers
					~~.org
				edu DNS Servers
					~~.edu
